Use the following endpoints to interact with the API:

  • GET /list - Get a list of restaurants
  • GET /detail/<id> - Get details of a specific restaurant
  • GET /search?q=<query> - Search for restaurants
  • POST /review - Add a review to a restaurant
  • PUT /review/update - Update a review of a restaurant
  • DELETE /review/delete - Delete a review from a restaurant
